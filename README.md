# Firefox DL Path Manager

Firefox のダウンロード履歴を一覧表示し、保存先パスやURLを素早くコピーするためのFirefox限定WebExtensionです。

## 機能

- ツールバーボタンから最近のダウンロード一覧を表示
- ファイルごとの保存先パス、URLコピー
- ファイル名、URL、保存先パスで絞り込み
- 件数と状態で絞り込み
- 完了済みファイルの階層を開く
- 完了済みファイルを実行
- ポップアップ上の `×` ボタンで閉じる

## Firefoxで読み込む

1. Firefox で `about:debugging#/runtime/this-firefox` を開く
2. `一時的なアドオンを読み込む...` を押す
3. このフォルダの `manifest.json` を選択する
4. ツールバーの `DL Path Manager` ボタンを開く

## 注意

- 取得できるのはFirefoxのダウンロード履歴に残っている項目です。
- 履歴を消した項目やプライベートウィンドウ由来の履歴は表示できないことがあります。
- ファイルを手動で移動した場合、Firefox履歴上の保存先パスは古いままの場合があります。

## プライバシー

この拡張機能は外部サーバーへ通信しません。ダウンロード履歴の表示とコピー処理は、Firefox内のWebExtension APIとローカルのクリップボード操作だけで完結します。
